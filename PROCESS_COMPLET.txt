## Local-first MVP architecture summary (Windows + VS Code, CPU-only)

RheumaAssist V1 is a **local-first clinical prototype** designed to run on a **CPU-only Windows laptop** with minimal infrastructure. It is intentionally scoped to match the **validated research reality**: a **PyTorch ResNet‑18 binary erosion detector** operating on **wrist ROI images (224×224)**, producing an **erosion probability and binary prediction** that must be **reviewed and signed by a clinician**. The application **never claims to diagnose rheumatoid arthritis** and always separates: **AI result → clinician review → clinician interpretation**.

**Frontend (Next.js 14 + TS + Tailwind)**  
Runs locally via `npm run dev`. Provides:
- ROI-direct upload (validated workflow)
- Full radiograph upload + **manual ROI crop** (practical workflow) with explicit warnings about ROI quality
- Patient registry, patient profile, longitudinal timeline, comparison view, assistant panel, admin screens

**Backend (FastAPI + SQLAlchemy + Alembic + SQLite)**  
Runs locally via `python -m uvicorn ...` and stores state in a local **SQLite** DB file.  
- Uses **cookie-based JWT session** with recommended cookie attributes and a minimal CSRF defense for state-changing requests. citeturn1search4  
- Uses **Argon2 password hashing**, aligned with OWASP guidance for password storage. citeturn0search2  
- Validates uploads using an allowlist + image verification and safe storage, aligned with OWASP upload guidance. citeturn0search3turn0search7  

**AI inference (PyTorch CPU-only)**  
- Loads the real artifacts from `model_artifacts/`  
- Loads weights using the `state_dict` pattern recommended by PyTorch documentation. citeturn0search1turn0search5  
- Optionally loads TorchScript if present  
- Performs required preprocessing: grayscale → 3-channel → resize 224×224 → normalize → sigmoid → threshold

**Background processing**  
V1 runs inference inline (simple) and remains structured so you can later add background tasks if needed; FastAPI supports background task patterns when you decide to offload processing. citeturn0search0

# RheumaAssist (Local-first MVP)

RheumaAssist is a local-first, English-language web application designed for internal hospital use to support **AI-assisted early radiographic assessment** in rheumatology workflows.

## Clinical safety and scope (important)

This application is **not** an autonomous diagnosis tool.

- The AI component in version 1 is a **CPU-only PyTorch model** that predicts **binary erosion presence** from **wrist ROI images** (224×224).
- The model was trained on **joint/bone-level ROI images**, not on full raw radiographs.
- The system provides an **assistive probability score and prediction** that must be reviewed by a licensed clinician.
- The application explicitly separates:
  - AI result
  - clinician review
  - final clinician interpretation

Required disclaimers are visible throughout the UI:
- “AI output is assistive and must be reviewed by a licensed clinician.”
- “This system supports early radiographic assessment and does not independently diagnose rheumatoid arthritis.”
- “Prediction quality depends on image quality and ROI selection.”
- “Final interpretation remains the responsibility of the clinician.”

## Local-first goals

This repository is intentionally optimized for:
- Windows laptop development
- VS Code
- CPU-only inference
- SQLite database
- No Docker required
- No Redis required
- No PostgreSQL required

You can run locally with:
- Backend: `python -m uvicorn app.main:app --reload`
- Frontend: `npm run dev`

## Troubleshooting (Windows)

### PowerShell execution policy error when activating venv
If you get an activation policy error, run:

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Then try:
```powershell
.\.venv\Scripts\Activate.ps1
```

### Torch installation issues on Windows
If `pip install torch` fails, install from the official PyTorch selector instructions for your Python version and CPU build, then re-run:
```powershell
pip install -r requirements.txt
```

### Cookies not set / login not working
- Ensure backend runs on `127.0.0.1:8000`
- Ensure frontend `.env.local` uses `NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8000/api`
- Ensure your browser allows cookies for localhost

---

## Security notes (local MVP)

- Passwords are hashed (Argon2).
- Auth uses HttpOnly cookies.
- Uploads are validated, size-limited, and stored privately.
- RBAC is enforced on sensitive endpoints.
- Audit logs are recorded in SQLite.

---

## License / internal use

This is a hospital-oriented prototype for internal use. Do not deploy externally without appropriate governance, security review, and compliance controls.

```python
from logging.config import fileConfig

from alembic import context
from sqlalchemy import engine_from_config, pool

from app.core.config import settings
from app.db.base import Base
from app.models import (  # noqa: F401
    user,
    role,
    patient,
    exam,
    uploaded_image,
    roi_selection,
    ai_analysis,
    clinician_review,
    assistant,
    audit_log,
)

config = context.config
fileConfig(config.config_file_name)

target_metadata = Base.metadata


def get_url() -> str:
    return settings.database_url


def run_migrations_offline() -> None:
    context.configure(
        url=get_url(),
        target_metadata=target_metadata,
        literal_binds=True,
        compare_type=True,
        dialect_opts={"paramstyle": "named"},
    )
    with context.begin_transaction():
        context.run_migrations()


def run_migrations_online() -> None:
    configuration = config.get_section(config.config_ini_section) or {}
    configuration["sqlalchemy.url"] = get_url()

    connectable = engine_from_config(
        configuration,
        prefix="sqlalchemy.",
        poolclass=pool.NullPool,
    )

    with connectable.connect() as connection:
        context.configure(connection=connection, target_metadata=target_metadata, compare_type=True)
        with context.begin_transaction():
            context.run_migrations()


if context.is_offline_mode():
    run_migrations_offline()
else:
    run_migrations_online()
```

import secrets
from fastapi import Request
from starlette.middleware.base import BaseHTTPMiddleware
from starlette.responses import JSONResponse

from app.core.config import settings


SAFE_METHODS = {"GET", "HEAD", "OPTIONS"}


def generate_csrf_token() -> str:
    return secrets.token_urlsafe(32)


class CSRFMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request: Request, call_next):
        if request.method.upper() in SAFE_METHODS:
            return await call_next(request)

        # Only enforce CSRF if an auth cookie exists.
        auth_cookie = request.cookies.get(settings.cookie_name)
        if not auth_cookie:
            return await call_next(request)

        csrf_cookie = request.cookies.get(settings.csrf_cookie_name)
        csrf_header = request.headers.get(settings.csrf_header_name)

        if not csrf_cookie or not csrf_header or csrf_cookie != csrf_header:
            return JSONResponse(
                status_code=403,
                content={"error": {"code": "csrf_failed", "message": "CSRF validation failed."}},
            )

        return await call_next(request)

FILE: rheumaassist/backend/app/core/rate_limit.py
```python
import time
from dataclasses import dataclass
from threading import Lock


@dataclass
class RateLimitResult:
    allowed: bool
    remaining: int
    reset_seconds: int


class InMemoryRateLimiter:
    """
    Minimal in-memory fixed-window limiter for local MVP.
    Not suitable for multi-process production use.
    """

    def __init__(self) -> None:
        self._lock = Lock()
        self._buckets: dict[str, dict[int, int]] = {}

    def check(self, key: str, limit: int, window_seconds: int) -> RateLimitResult:
        now = int(time.time())
        window = now // window_seconds

        with self._lock:
            win_map = self._buckets.setdefault(key, {})
            count = win_map.get(window, 0) + 1
            win_map[window] = count

            # small memory cleanup
            for w in list(win_map.keys()):
                if w < window - 2:
                    del win_map[w]

        remaining = max(0, limit - count)
        allowed = count <= limit
        reset_seconds = window_seconds - (now % window_seconds)
        return RateLimitResult(allowed=allowed, remaining=remaining, reset_seconds=reset_seconds)

class PasswordService:
    def __init__(self) -> None:
        self._ctx = CryptContext(schemes=["argon2"], deprecated="auto")

    def hash_password(self, password: str) -> str:
        if len(password) < 10:
            raise AppError("Password must be at least 10 characters long.", code="weak_password", http_status=400)
        return self._ctx.hash(password)

    def verify_password(self, password: str, password_hash: str) -> bool:
        try:
            return self._ctx.verify(password, password_hash)
        except Exception:
            return False

from datetime import datetime, timedelta, timezone

from fastapi import Response
from sqlalchemy.orm import Session

from app.core.config import settings
from app.core.csrf import generate_csrf_token
from app.core.errors import AuthError
from app.models.user import User
from app.services.password_service import PasswordService
from app.services.token_service import TokenService


class AuthService:
    def __init__(self) -> None:
        self._passwords = PasswordService()
        self._tokens = TokenService()

    def login(self, db: Session, *, email: str, password: str, remember_me: bool, response: Response) -> User:
        user = db.query(User).filter(User.email == email.lower().strip()).first()
        if not user or not user.is_active:
            raise AuthError("Invalid credentials.", http_status=401)

        # lockout
        if user.locked_until and user.locked_until.replace(tzinfo=timezone.utc) > datetime.now(timezone.utc):
            raise AuthError("Account locked. Try again later.", http_status=403)

        if not self._passwords.verify_password(password, user.password_hash):
            user.failed_login_attempts += 1
            if user.failed_login_attempts >= 8:
                user.locked_until = datetime.now(timezone.utc) + timedelta(minutes=15)
                user.failed_login_attempts = 0
            db.add(user)
            db.commit()
            raise AuthError("Invalid credentials.", http_status=401)

        user.failed_login_attempts = 0
        user.locked_until = None
        user.last_login_at = datetime.utcnow()
        db.add(user)
        db.commit()

        roles = [r.name for r in user.roles]

        access = self._tokens.create_access(user.id, roles)

        ttl_minutes = settings.access_token_ttl_minutes
        if remember_me:
            ttl_minutes = max(ttl_minutes, 24 * 60)

        response.set_cookie(
            settings.cookie_name,
            access,
            httponly=True,
            secure=settings.cookie_secure,
            samesite=settings.cookie_samesite,
            max_age=ttl_minutes * 60,
            path="/",
        )

        # Double-submit CSRF token
        csrf = generate_csrf_token()
        response.set_cookie(
            settings.csrf_cookie_name,
            csrf,
            httponly=False,
            secure=settings.cookie_secure,
            samesite=settings.cookie_samesite,
            max_age=ttl_minutes * 60,
            path="/",
        )

        return user

    def logout(self, response: Response) -> None:
        response.delete_cookie(settings.cookie_name, path="/")
        response.delete_cookie(settings.csrf_cookie_name, path="/")

import hashlib
import io
from pathlib import Path
from uuid import uuid4

from PIL import Image as PILImage

from app.core.config import settings
from app.core.errors import AppError


class StoredFile:
    def __init__(self, *, stored_filename: str, storage_path: str, sha256: str, mime_type: str, size_bytes: int, width: int, height: int):
        self.stored_filename = stored_filename
        self.storage_path = storage_path
        self.sha256 = sha256
        self.mime_type = mime_type
        self.size_bytes = size_bytes
        self.width = width
        self.height = height


class StorageService:
    def __init__(self) -> None:
        self._uploads_dir = settings.resolve_path(settings.uploads_dir)
        self._uploads_dir.mkdir(parents=True, exist_ok=True)

        allowed = [x.strip() for x in settings.allowed_image_types.split(",") if x.strip()]
        self._allowed_mime = set(allowed)

    def store_image_bytes(self, *, original_filename: str, data: bytes) -> StoredFile:
        if len(data) > settings.max_upload_mb * 1024 * 1024:
            raise AppError("File too large.", code="file_too_large", http_status=413)

        # Validate by decoding with PIL + verify format
        try:
            img = PILImage.open(io.BytesIO(data))
            img.verify()
        except Exception:
            raise AppError("Invalid image file.", code="invalid_image", http_status=400)

        img = PILImage.open(io.BytesIO(data))
        fmt = (img.format or "").upper()
        if fmt not in {"PNG", "JPEG", "JPG"}:
            raise AppError("Only PNG and JPEG are supported in V1.", code="unsupported_format", http_status=400)

        mime = "image/png" if fmt == "PNG" else "image/jpeg"
        if mime not in self._allowed_mime:
            raise AppError("Unsupported MIME type.", code="unsupported_mime", http_status=400)

        width, height = img.size
        sha256 = hashlib.sha256(data).hexdigest()

        ext = "png" if mime == "image/png" else "jpg"
        stored_filename = f"{uuid4()}.{ext}"

        dest = (self._uploads_dir / stored_filename).resolve()
        if not str(dest).startswith(str(self._uploads_dir.resolve())):
            raise AppError("Invalid storage path.", code="invalid_storage_path", http_status=400)

        dest.write_bytes(data)

        return StoredFile(
            stored_filename=stored_filename,
            storage_path=str(dest),
            sha256=sha256,
            mime_type=mime,
            size_bytes=len(data),
            width=width,
            height=height,
        )

import torch
import torch.nn as nn
import torchvision.models as models


class EROBinaryResNet18(nn.Module):
    def __init__(self, dropout: float = 0.6):
        super().__init__()
        self.backbone = models.resnet18(weights=None)
        in_features = self.backbone.fc.in_features
        self.backbone.fc = nn.Sequential(
            nn.Dropout(dropout),
            nn.Linear(in_features, 1),
        )

    def forward(self, x: torch.Tensor) -> torch.Tensor:
        return self.backbone(x).squeeze(1)

import json
from dataclasses import dataclass
from pathlib import Path
from typing import Any

import torch

from app.core.config import settings
from app.core.errors import AppError
from app.ai.model_arch import EROBinaryResNet18


@dataclass
class LoadedModel:
    model: Any  # torch.nn.Module or torch.jit.ScriptModule
    config: dict[str, Any]


_cached: LoadedModel | None = None


def _path(base_dir: str, filename: str) -> Path:
    base = settings.resolve_path(base_dir)
    return (base / filename).resolve()


def load_model() -> LoadedModel:
    global _cached
    if _cached is not None:
        return _cached

    cfg_path = _path(settings.model_artifacts_dir, settings.model_config_filename)
    ckpt_path = _path(settings.model_artifacts_dir, settings.model_checkpoint_filename)
    ts_path = _path(settings.model_artifacts_dir, settings.model_scripted_filename)

    if not cfg_path.exists():
        raise AppError(f"Model config missing: {cfg_path.name}", code="model_config_missing", http_status=500)

    cfg = json.loads(cfg_path.read_text(encoding="utf-8"))
    device = torch.device("cpu")

    if settings.use_torchscript:
        if not ts_path.exists():
            raise AppError("TorchScript requested but file is missing.", code="model_torchscript_missing", http_status=500)
        model = torch.jit.load(str(ts_path), map_location=device)
        model.eval()
        _cached = LoadedModel(model=model, config=cfg)
        return _cached

    if not ckpt_path.exists():
        raise AppError(f"Model checkpoint missing: {ckpt_path.name}", code="model_checkpoint_missing", http_status=500)

    # Safer load when supported by the installed PyTorch version
    try:
        checkpoint = torch.load(str(ckpt_path), map_location=device, weights_only=True)
    except TypeError:
        checkpoint = torch.load(str(ckpt_path), map_location=device)

    dropout = float(checkpoint.get("dropout", cfg.get("dropout", 0.6)))
    model = EROBinaryResNet18(dropout=dropout)

    state = checkpoint.get("state_dict")
    if not state:
        raise AppError("Checkpoint does not contain state_dict.", code="invalid_checkpoint", http_status=500)

    model.load_state_dict(state)
    model.to(device)
    model.eval()

    merged = dict(cfg)
    merged["threshold"] = float(checkpoint.get("threshold", cfg.get("threshold", 0.314)))
    merged["train_mean"] = checkpoint.get("train_mean", cfg.get("train_mean", [0.0, 0.0, 0.0]))
    merged["train_std"] = checkpoint.get("train_std", cfg.get("train_std", [1.0, 1.0, 1.0]))
    merged["model_version"] = checkpoint.get("model_class", cfg.get("model_class", "EROBinaryResNet18"))

    _cached = LoadedModel(model=model, config=merged)
    return _cached

from typing import Any
from PIL import Image
import torch
from torchvision import transforms


def preprocess_image(path: str, *, mean: list[float], std: list[float]) -> tuple[torch.Tensor, dict[str, Any]]:
    img = Image.open(path)

    # Enforce consistent preprocessing:
    # - grayscale if needed
    # - convert to 3 channels
    img = img.convert("L").convert("RGB")

    tf = transforms.Compose(
        [
            transforms.Resize((224, 224)),
            transforms.ToTensor(),
            transforms.Normalize(mean=mean, std=std),
        ]
    )
    x = tf(img).unsqueeze(0)  # (1,3,224,224)

    meta = {"conversion": "L->RGB", "resize": [224, 224]}
    return x, meta

from typing import Any
import torch

from app.ai.model_loader import load_model
from app.ai.preprocess import preprocess_image


@torch.no_grad()
def infer_erosion(image_path: str) -> dict[str, Any]:
    loaded = load_model()
    cfg = loaded.config

    threshold = float(cfg.get("threshold", 0.314))
    mean = cfg.get("train_mean", [0.0, 0.0, 0.0])
    std = cfg.get("train_std", [1.0, 1.0, 1.0])

    x, preprocess_meta = preprocess_image(image_path, mean=mean, std=std)
    x = x.to("cpu")

    logits = loaded.model(x)
    prob = torch.sigmoid(logits).item()
    pred = 1 if prob >= threshold else 0

    return {
        "probability": float(prob),
        "predicted_class": int(pred),
        "threshold": float(threshold),
        "model_version": str(cfg.get("model_version", "EROBinaryResNet18")),
        "preprocess": preprocess_meta,
    }

from datetime import datetime, timezone

from fastapi import Depends, Request
from sqlalchemy.orm import Session

from app.core.config import settings
from app.core.errors import AuthError, ForbiddenError
from app.core.security import decode_token
from app.db.deps import get_db
from app.models.user import User


class AuthDeps:
    @staticmethod
    def require_user(request: Request, db: Session = Depends(get_db)) -> User:
        token = request.cookies.get(settings.cookie_name)
        if not token:
            raise AuthError("Not authenticated.", http_status=401)

        try:
            payload = decode_token(token)
        except Exception:
            raise AuthError("Invalid session.", http_status=401)

        if payload.get("type") != "access":
            raise AuthError("Invalid session type.", http_status=401)

        user_id = payload.get("sub")
        if not user_id:
            raise AuthError("Invalid session payload.", http_status=401)

        user = db.query(User).filter(User.id == user_id).first()
        if not user or not user.is_active:
            raise AuthError("Account disabled.", http_status=401)

        if user.locked_until and user.locked_until.replace(tzinfo=timezone.utc) > datetime.now(timezone.utc):
            raise AuthError("Account locked. Try again later.", http_status=403)

        return user

    @staticmethod
    def require_roles(*roles: str):
        def _dep(user: User = Depends(AuthDeps.require_user)) -> User:
            if any(user.has_role(r) for r in roles):
                return user
            raise ForbiddenError("Insufficient privileges.", http_status=403)

        return _dep

import json
from datetime import datetime, date
from uuid import uuid4

from fastapi import UploadFile
from sqlalchemy.orm import Session

from app.ai.inference import infer_erosion
from app.core.errors import AppError, NotFoundError
from app.models.patient import Patient
from app.models.exam import Exam
from app.models.uploaded_image import UploadedImage
from app.models.roi_selection import ROISelection

from app.services.storage_service import StorageService
from app.models.user import User


class AnalysisService:
    def create_and_run(
        self,
        db: Session,
        *,
        actor_user: User,
        patient_public_id: str,
        exam_date_iso: str,
        mode: str,
        roi_image: UploadFile,
        full_image: UploadFile | None,
        roi_meta: dict,
    ) -> AIAnalysis:
        patient = db.query(Patient).filter(Patient.patient_id == patient_public_id).first()
        if not patient:
            raise NotFoundError("Patient not found.", http_status=404)

        try:
            exam_date_obj = date.fromisoformat(exam_date_iso)
        except Exception:
            raise AppError("Invalid exam_date. Use YYYY-MM-DD.", code="invalid_exam_date", http_status=400)

        if mode not in {"ROI_DIRECT", "FULL_WITH_MANUAL_ROI"}:
            raise AppError("Invalid mode.", code="invalid_mode", http_status=400)

        if mode == "FULL_WITH_MANUAL_ROI" and full_image is None:
            raise AppError("Full radiograph image is required for FULL_WITH_MANUAL_ROI.", code="missing_full_image", http_status=400)

        now = datetime.utcnow()

        exam = Exam(
            id=str(uuid4()),
            patient_id=patient.id,
            exam_date=exam_date_obj,
            created_by_user_id=actor_user.id,
            notes=None,
            created_at=now,
        )
        db.add(exam)
        db.commit()
        db.refresh(exam)

        storage = StorageService()

        roi_bytes = roi_image.file.read()
        roi_store = storage.store_image_bytes(original_filename=roi_image.filename or "roi", data=roi_bytes)

        roi_rec = UploadedImage(
            id=str(uuid4()),
            patient_id=patient.id,
            exam_id=exam.id,
            image_kind="ROI_DIRECT" if mode == "ROI_DIRECT" else "ROI_CROPPED",
            original_filename=roi_image.filename or "roi",
            stored_filename=roi_store.stored_filename,
            storage_path=roi_store.storage_path,
            sha256=roi_store.sha256,
            mime_type=roi_store.mime_type,
            size_bytes=roi_store.size_bytes,
            width=roi_store.width,
            height=roi_store.height,
            uploaded_by_user_id=actor_user.id,
            uploaded_at=now,
        )
        db.add(roi_rec)
        db.commit()
        db.refresh(roi_rec)

        full_rec = None
        roi_sel = None

        if mode == "FULL_WITH_MANUAL_ROI" and full_image is not None:
            full_bytes = full_image.file.read()
            full_store = storage.store_image_bytes(original_filename=full_image.filename or "full", data=full_bytes)

            full_rec = UploadedImage(
                id=str(uuid4()),
                patient_id=patient.id,
                exam_id=exam.id,
                image_kind="FULL_RADIOGRAPH",
                original_filename=full_image.filename or "full",
                stored_filename=full_store.stored_filename,
                storage_path=full_store.storage_path,
                sha256=full_store.sha256,
                mime_type=full_store.mime_type,
                size_bytes=full_store.size_bytes,
                width=full_store.width,
                height=full_store.height,
                uploaded_by_user_id=actor_user.id,
                uploaded_at=now,
            )
            db.add(full_rec)
            db.commit()
            db.refresh(full_rec)

            if all(roi_meta.get(k) is not None for k in ["x", "y", "width", "height"]):
                roi_sel = ROISelection(
                    id=str(uuid4()),
                    source_image_id=full_rec.id,
                    roi_image_id=roi_rec.id,
                    x=int(roi_meta["x"]),
                    y=int(roi_meta["y"]),
                    width=int(roi_meta["width"]),
                    height=int(roi_meta["height"]),
                    created_by_user_id=actor_user.id,
                    created_at=now,
                )
                db.add(roi_sel)
                db.commit()
                db.refresh(roi_sel)

        analysis = AIAnalysis(
            id=str(uuid4()),
            patient_id=patient.id,
            exam_id=exam.id,
            roi_image_id=roi_rec.id,
            full_image_id=full_rec.id if full_rec else None,
            roi_selection_id=roi_sel.id if roi_sel else None,
            status="RUNNING",
            erosion_probability=None,
            threshold=None,
            predicted_class=None,
            model_version=None,
            error_message=None,
            created_by_user_id=actor_user.id,
            created_at=now,
            started_at=now,
            completed_at=None,
            metadata_json=json.dumps(
                {
                    "mode": mode,
                    "warnings": ["Prediction quality depends on image quality and ROI selection."]
                    if mode == "FULL_WITH_MANUAL_ROI"
                    else [],
                }
            ),
        )
        db.add(analysis)
        db.commit()
        db.refresh(analysis)

        # Run real inference on CPU (no fake output)
        try:
            res = infer_erosion(roi_rec.storage_path)
            analysis.status = "SUCCESS"
            analysis.erosion_probability = float(res["probability"])
            analysis.threshold = float(res["threshold"])
            analysis.predicted_class = int(res["predicted_class"])
            analysis.model_version = str(res["model_version"])
            analysis.completed_at = datetime.utcnow()
            analysis.metadata_json = json.dumps(
                {
                    **json.loads(analysis.metadata_json),
                    "preprocess": res["preprocess"],
                }
            )
            db.add(analysis)
            db.commit()
        except AppError as e:
            analysis.status = "FAILED"
            analysis.error_message = e.message
            analysis.completed_at = datetime.utcnow()
            db.add(analysis)
            db.commit()
            raise
        except Exception:
            analysis.status = "FAILED"
            analysis.error_message = "Inference failed. Verify model artifacts and image validity."
            analysis.completed_at = datetime.utcnow()
            db.add(analysis)
            db.commit()
            raise AppError(analysis.error_message, code="inference_failed", http_status=500)

        return analysis

    def get_detail(self, db: Session, *, analysis_id: str):
        a = db.query(AIAnalysis).filter(AIAnalysis.id == analysis_id).first()
        if not a:
            raise NotFoundError("Analysis not found.", http_status=404)

        exam = db.query(Exam).filter(Exam.id == a.exam_id).first()
        patient = db.query(Patient).filter(Patient.id == a.patient_id).first()

        return {
            "id": a.id,
            "status": a.status,
            "patient": {
                "id": patient.id,
                "patient_id": patient.patient_id,
                "full_name": patient.full_name,
            } if patient else None,
            "exam": {
                "id": exam.id,
                "exam_date": str(exam.exam_date),
            } if exam else None,
            "erosion_probability": a.erosion_probability,
            "threshold": a.threshold,
            "predicted_class": a.predicted_class,
            "model_version": a.model_version,
            "created_at": a.created_at,
            "started_at": a.started_at,
            "completed_at": a.completed_at,
            "roi_image_id": a.roi_image_id,
            "full_image_id": a.full_image_id,
            "roi_selection_id": a.roi_selection_id,
            "error_message": a.error_message,
            "metadata": json.loads(a.metadata_json or "{}"),
        }

    def list_by_patient_public_id(self, db: Session, *, patient_public_id: str):
        p = db.query(Patient).filter(Patient.patient_id == patient_public_id).first()
        if not p:
            raise NotFoundError("Patient not found.", http_status=404)

        items = (
            db.query(AIAnalysis)
            .filter(AIAnalysis.patient_id == p.id)
            .order_by(AIAnalysis.created_at.desc())
            .all()
        )

        out = []
        for a in items:
            exam = db.query(Exam).filter(Exam.id == a.exam_id).first()
            out.append(
                {
                    "id": a.id,
                    "status": a.status,
                    "exam_date": str(exam.exam_date) if exam else None,
                    "erosion_probability": a.erosion_probability,
                    "predicted_class": a.predicted_class,
                    "model_version": a.model_version,
                    "created_at": a.created_at,
                }
            )

        return {"patient_id": p.patient_id, "items": out}

class ReviewService:
    def _get_analysis(self, db: Session, analysis_id: str) -> AIAnalysis:

        if not a:

    def _current_review(self, db: Session, analysis_id: str) -> ClinicianReview | None:
        return (
            db.query(ClinicianReview)
            .filter(ClinicianReview.analysis_id == analysis_id, ClinicianReview.is_current == True)  # noqa: E712
            .first()
        )

    def save_draft(self, db: Session, *, analysis_id: str, user: User, payload: ReviewCreateRequest) -> ReviewResponse:
        a = self._get_analysis(db, analysis_id)
        current = self._current_review(db, analysis_id)

        now = datetime.utcnow()
        version = 1
        if current:
            current.is_current = False
            db.add(current)
            db.commit()
            version = current.version + 1

        r = ClinicianReview(
            id=str(uuid4()),
            analysis_id=a.id,
            patient_id=a.patient_id,
            reviewer_user_id=user.id,
            status="DRAFT",
            decision=payload.decision,
            structured_note_json=json.dumps(payload.structured_note or {}, ensure_ascii=False),
            free_text=payload.free_text,
            version=version,
            is_current=True,
            signed_at=None,
            created_at=now,
        )
        db.add(r)
        db.commit()
        db.refresh(r)

        return self._to_response(r)

    def finalize(self, db: Session, *, analysis_id: str, user: User, payload: ReviewCreateRequest) -> ReviewResponse:
        a = self._get_analysis(db, analysis_id)

        now = datetime.utcnow()
        current = self._current_review(db, analysis_id)
        version = 1
        if current:
            if current.status == "FINAL":
                raise ConflictError("A final review already exists. Create a new version via draft first.", http_status=409)
            current.is_current = False
            db.add(current)
            db.commit()
            version = current.version + 1

        r = ClinicianReview(
            id=str(uuid4()),
            analysis_id=a.id,
            patient_id=a.patient_id,
            reviewer_user_id=user.id,
            status="FINAL",
            decision=payload.decision,
            structured_note_json=json.dumps(payload.structured_note or {}, ensure_ascii=False),
            free_text=payload.free_text,
            version=version,
            is_current=True,
            signed_at=now,
            created_at=now,
        )
        db.add(r)
        db.commit()
        db.refresh(r)

        return self._to_response(r)

    def _to_response(self, r: ClinicianReview) -> ReviewResponse:
        return ReviewResponse(
            id=r.id,
            analysis_id=r.analysis_id,
            patient_id=r.patient_id,
            reviewer_user_id=r.reviewer_user_id,
            status=r.status,
            decision=r.decision,
            structured_note=json.loads(r.structured_note_json or "{}"),
            free_text=r.free_text,
            version=r.version,
            is_current=r.is_current,
            signed_at=r.signed_at,
            created_at=r.created_at,
        )

```python
import re
from datetime import date, timedelta
from sqlalchemy.orm import Session

from app.core.errors import NotFoundError
from app.models.patient import Patient
from app.models.ai_analysis import AIAnalysis
from app.models.exam import Exam
from app.models.user import User
from app.schemas.assistant import AssistantQueryResponse, AssistantCitation


class AssistantService:
    """
    Local-first grounded assistant.

    - No public internet.
    - Answers only from stored patient data and analyses.
    - Always returns citations (analysis IDs + exam dates used).
    """

    def answer(self, db: Session, *, patient_public_id: str, user: User, question: str) -> AssistantQueryResponse:
        patient = db.query(Patient).filter(Patient.patient_id == patient_public_id).first()
        if not patient:
            raise NotFoundError("Patient not found.", http_status=404)

        analyses = (
            db.query(AIAnalysis)
            .filter(AIAnalysis.patient_id == patient.id, AIAnalysis.status == "SUCCESS")
            .order_by(AIAnalysis.created_at.asc())
            .all()
        )

        if not analyses:
            return AssistantQueryResponse(
                answer="Insufficient data: no successful AI analyses are stored for this patient yet.",
                citations=[],
                used_records_count=0,
            )

        q = question.strip().lower()

        # Helper: attach exam dates
        def exam_date_for(a: AIAnalysis) -> str | None:
            e = db.query(Exam).filter(Exam.id == a.exam_id).first()
            return str(e.exam_date) if e else None

        # Detect "weeks ago" compare request
        m = re.search(r"(\d+)\s*weeks?\s*ago", q)
        if "compare" in q and m:
            weeks = int(m.group(1))
            latest = analyses[-1]
            latest_date_str = exam_date_for(latest)
            latest_date = date.fromisoformat(latest_date_str) if latest_date_str else None

            if not latest_date:
                return AssistantQueryResponse(
                    answer="Insufficient data: latest exam date is missing.",
                    citations=[AssistantCitation(analysis_id=latest.id, exam_date=latest_date_str)],
                    used_records_count=1,
                )

            target = latest_date - timedelta(weeks=weeks)

            # Choose closest analysis by exam_date
            best = None
            best_delta_days = None
            for a in analyses:
                d_str = exam_date_for(a)
                if not d_str:
                    continue
                d = date.fromisoformat(d_str)
                delta = abs((d - target).days)
                if best is None or delta < best_delta_days:
                    best = a
                    best_delta_days = delta

            if not best:
                return AssistantQueryResponse(
                    answer="Insufficient data: could not find a comparable prior exam date.",
                    citations=[AssistantCitation(analysis_id=latest.id, exam_date=latest_date_str)],
                    used_records_count=1,
                )

            delta = (latest.erosion_probability or 0.0) - (best.erosion_probability or 0.0)

            answer = (
                f"Comparison based on stored analyses only:\n"
                f"- Latest exam ({latest_date_str}): erosion probability = {latest.erosion_probability:.3f}\n"
                f"- Reference exam (~{weeks} weeks ago, {exam_date_for(best)}): erosion probability = {best.erosion_probability:.3f}\n"
                f"- Delta (latest - reference) = {delta:.3f}\n\n"
                f"Note: This is an assistive score comparison and does not diagnose rheumatoid arthritis."
            )

            return AssistantQueryResponse(
                answer=answer,
                citations=[
                    AssistantCitation(analysis_id=latest.id, exam_date=latest_date_str),
                    AssistantCitation(analysis_id=best.id, exam_date=exam_date_for(best)),
                ],
                used_records_count=2,
            )

        if "summarize" in q and ("imaging history" in q or "history" in q):
            lines = []
            citations = []
            for a in analyses:
                d = exam_date_for(a)
                lines.append(f"- {d}: probability={a.erosion_probability:.3f}, class={a.predicted_class}")
                citations.append(AssistantCitation(analysis_id=a.id, exam_date=d))
            answer = (
                "Imaging history summary (stored analyses only):\n"
                + "\n".join(lines)
                + "\n\nReminder: AI output is assistive and must be reviewed by a licensed clinician."
            )
            return AssistantQueryResponse(answer=answer, citations=citations, used_records_count=len(citations))

        if "increased" in q and ("over time" in q or "probability" in q):
            first = analyses[0]
            last = analyses[-1]
            d_first = exam_date_for(first)
            d_last = exam_date_for(last)
            delta = (last.erosion_probability or 0.0) - (first.erosion_probability or 0.0)

            trend = "increased" if delta > 0.02 else "decreased" if delta < -0.02 else "remained approximately stable"

            answer = (
                f"Trend based on stored analyses only:\n"
                f"- First stored exam ({d_first}): probability={first.erosion_probability:.3f}\n"
                f"- Latest stored exam ({d_last}): probability={last.erosion_probability:.3f}\n"
                f"- Net change = {delta:.3f} → probability has {trend}.\n\n"
                f"Reminder: This is an assistive score trend and does not diagnose rheumatoid arthritis."
            )

            return AssistantQueryResponse(
                answer=answer,
                citations=[
                    AssistantCitation(analysis_id=first.id, exam_date=d_first),
                    AssistantCitation(analysis_id=last.id, exam_date=d_last),
                ],
                used_records_count=2,
            )

        if "show" in q and ("analyses" in q or "previous analyses" in q):
            lines = []
            citations = []
            for a in analyses:
                d = exam_date_for(a)
                lines.append(f"- {d}: probability={a.erosion_probability:.3f}, class={a.predicted_class}")
                citations.append(AssistantCitation(analysis_id=a.id, exam_date=d))
            answer = "Stored AI analyses:\n" + "\n".join(lines)
            return AssistantQueryResponse(answer=answer, citations=citations, used_records_count=len(citations))

        # Default grounded response
        return AssistantQueryResponse(
            answer=(
                "I can only answer using stored patient records and prior analyses.\n"
                "Try asking:\n"
                "- “Summarize this patient’s imaging history.”\n"
                "- “Compare today’s result with the exam from 12 weeks ago.”\n"
                "- “Has the erosion probability increased over time?”"
            ),
            citations=[],
            used_records_count=0,
        )
